# CyberDefender Labs

A collection of hands-on labs completed as part of the **[CyberDefender](https://cyberdefenders.org/)** training program. Each lab folder includes the objective, tools used, skills gained, and a write-up of the approach taken.

## About

CyberDefender is a cybersecurity training platform offering practical, scenario-based labs across areas such as network security, malware analysis, digital forensics, and incident response. This repo documents my progress through the program as a portfolio of applied, hands-on work.
